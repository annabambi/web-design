Taking Type to the Web

Knowing the usage of the fonts is very important skill that every designer should have for creating beautiful web pages. There are nine font families that are most common and supported by the both major operating systems (see picture below). Those families are called the safe list. The font-family of CSS allows you to choose differents fonts in the order of its preference (this is referred to as a font stack). Any user can use the fonts for the future designs.





Text Image Replacement

If you want to the visitors to see some text in a specific typeface you should to replace the text with an image. You can mark up your text as text and then use CSS to create a link the size of your image. Also, you can try to use the Phark Method of image replacement which involves setting the image that you want to use as the background of the element. After this step, you should add a negative text-indent value high enough to move the text content out of the block.



Text Alignment

There are four ways how the text on your web page can be located:

Left aligned - the text is located on the left side of the page, the “ragged edge” is on the right.

Right Aligned - the text is located on the right side of the page, the “ragged edge” is on the left.

Centered - text is located at the center of the page. Centered text is usually used for document titles and may be appropriate for headers and footers.

Justified - the text combines left and right aligned text. By using justified text there could be big gaps between the words, so, the justification usually works well for the newspapers and the magazines.
