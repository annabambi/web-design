ANATOMY OF A WEB PAGE

Every web page has the same structure that helps to use site effectively and fast.

There are components of the web page:

1. Containing block is a main block that contain all components of the web page.

2. Header is a small area/block on the top of the page where you can fing logo, navigation bar and name of the creater.

3. Sidebar is a bar/block where all background information is located. The sidebar can be on the left, right or on the buttom above the footer. There are can be more than one sidebar on the page.

4. Main content is a block where all important and main informaion is located. Usually you can see it in the center of the web page.

5. Footer is a block that containts information like contacts, sponsors, copyright or button that allows you to go to the top of the page.

6. Backgroung is a color of the page that is located under the all components.

GRID THEORY

Grid system was created to help and navigate designers to create a layout of a web page. All elements should be proportional and do not block each other.

There are some examples how to use the grid system:

1. The Golden Ratio means that two quantities have the same ration as the ratio of their sum to the larger of the two quantities. We can take the whole lenght of the line and divide it by 1.62, so, we will get the lenght of the one of the sides of this line. This method helps designers and users to pay attention to the specific points on the web page and its interface.

2. The Rule of Third is similar to the Golden Ratio. The rule of third represents the whole surface of the web page that divided into thirds horizontally and vertically, and, also, divided by a line between each vertical line. The reqtangle in the center is the main part of the page that should have all important information or the main object.

3. 960 Grid System is a system that helps to organize layout for the web site. 960 px can be divided by 3, 4, 5, 6, 8, 10, 12, 15, and 16 which makes it ideal width for the grid. There are the layout foundation for the system: 12, 16 and 24 columns. 12 and 16 columns are more common. Those columns help to divide content into quarters.
